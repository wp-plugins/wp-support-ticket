Wordpress user support plugin. Registered users will be able to create new support tickets and reply to already created support tickets. 

== Description ==

* Registered users can create support tickets.
* Admin can reply on support tickets from admin panel.
* File can be attached to the reply message.
* Ticket can be marked as Open / Closed / Resolved from admin panel.


= Usage Shortcodes =
* [create_support] to display cretate new suppor ticket form.
* [ticket] to display tickets created by logged in user.
* [ticket_search] to display ticket search form.


== Installation ==

1. Upload `wp-support.zip` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to `Settings-> WP Support`, and set some options. It's really easy.
4. Use shortcodes or functions to display uploaded files in frontend.
